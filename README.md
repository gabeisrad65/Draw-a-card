#  Python Card Drawer

A terminal-based application that simulates a standard 52-card deck, allowing users to draw and visualize cards using ASCII art. 

##  Features
*   **Deck Management:** Generates a full 52-card deck using nested loops.
*   **Shuffling Logic:** Uses the `random` module to ensure a unique draw every time.
*   **ASCII Visualization:** Custom function to display cards in a classic visual format within the console.
*   **User Interaction:** Interactive command-line interface with error handling for user inputs.

##  Skills Demonstrated
*   **Python Fundamentals:** Functions, lists, and loops.
*   **Error Handling:** Using `try/except` blocks to prevent program crashes.
*   **Data Structures:** Using tuples to represent card ranks and suits.

##  How to Run
1. Ensure you have [Python](https://python.org) installed.
2. Download `draw.py`.
3. Run the following command in your terminal:
   ```bash
   python draw.py
   ```
